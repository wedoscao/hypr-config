# Hyprland Configuration

This repository contains my Hyprland configuration, built for speed, modern development, and productivity.

## Prerequisites

- hyprland
- hyprlock
- hypridle
- dolphin
- kitty
- walker
- waybar
- network-manager-applet
- cliphist
- wl-clipboard
- elephant
- awww
- hyprshot
- brightnessctl
- playerctl
- wireplumber
- xdg-desktop-portal-hyprland
- CaskaydiaCove Nerd Font

## How to install


Clone this repo into your Neovim config directory
```bash
   git clone https://https://github.com/wedoscao/hypr-config ~/.config/hypr
```
