# Hyprland Configuration

This repository contains my Hyprland configuration, built for speed, modern development, and productivity.

## Prerequisites

- hyprland
- hyprlock
- hypridle
- dolphin
- kitty
- walker
- waybar
- network-manager-applet
- cliphist
- wl-clipboard
- elephant
- awww
- hyprshot
- brightnessctl
- playerctl
- wireplumber
- xdg-desktop-portal-hyprland
- adw-gtk-theme
- qt6ct
- fcitx5
- hyprshutdown
- CaskaydiaCove Nerd Font

## How to install


1. Clone repo into your Hyprland config directory
```bash
git clone https://github.com/wedoscao/hypr-config ~/.config/hypr
```

2. Add your avatar image named avatar.png in this directory.

## Recommendation

Backup your old config first
```bash
mv ~/.config/hypr ~/.config/hypr-bak
```
